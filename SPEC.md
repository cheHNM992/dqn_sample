# 6×6オセロ（Reversi）ゲーム仕様
1. ゲームの概要

6×6マスの盤面で行う二人対戦型の完全情報ゲーム。
プレイヤーは交互に石を置き、相手の石を挟んで自分の色に反転させる。
最終的に盤上に自分の石が多いプレイヤーが勝利する。

2. 盤面

マス数：6行 × 6列（合計36マス）

各マスは以下のいずれかの状態を持つ：

黒石

白石

空

3. 初期配置

ゲーム開始時、盤面中央の4マスに以下のように石を配置する。

中央の2×2領域に：

左上：白

右上：黒

左下：黒

右下：白

（通常の8×8オセロと同じ対角配置）

4. プレイヤー

プレイヤーは2人

石の色は「黒」と「白」

黒が先手

5. 手番の進行

プレイヤーは交互に手を打つ

自分の手番では、以下の条件を満たすマスに石を置く

6. 合法手（石を置ける条件）

プレイヤーは、次の条件を満たす空マスにのみ石を置ける。

条件

そのマスが空である

8方向（上下左右＋斜め4方向）のいずれかで、

隣接マスに相手の石があり

その方向に連続して1個以上の相手石があり

その先に自分の石が存在する

このとき、間にある相手石をすべて自分の石に反転させる。

7. 石の反転

石を置いたとき、条件を満たす方向すべてにおいて

挟まれた相手石をすべて自分の色に裏返す

複数方向で同時に反転が起こる場合もある

8. パス

合法手が1つも存在しない場合、

プレイヤーは自動的にパスとなる

パス後、相手の手番になる

パスを任意に選ぶことはできない（合法手がある場合は必ず打つ）

9. ゲーム終了条件

次のいずれかを満たしたとき、ゲーム終了。

盤面がすべて埋まった

両プレイヤーが連続でパスした

10. 勝敗判定

ゲーム終了時、盤上の石の数を数える。

自分の石が相手より多い → 勝ち

少ない → 負け

同数 → 引き分け


# プログラム仕様
1. 対戦形式
下記の通り、人間が先攻と後攻の両方でプレーできる。
人間 vs CPU
CPU vs 人間

2. CPUアルゴリズム
6×6オセロ環境を実装し、DQNで自己対戦学習して強くなるエージェントを作る。

3. 自己対戦のログ
自己対戦のログを保存し、プログラム起動時に指定することで使用できる。

4. プログラム言語
Pythonを使用する。
